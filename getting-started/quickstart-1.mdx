## Prerequisites

- [Docker](https://docs.docker.com/get-docker/) installed (for production)
- [Node.js 20+](https://nodejs.org) and [pnpm](https://pnpm.io) (for local dev)
- A [Supabase](https://supabase.com) project — free tier works
- An [Anthropic API key](https://console.anthropic.com)

---

## Step 1: Clone and configure

```bash
git clone https://github.com/bennetglinder1/proply-crm.git
cd proply-crm
cp .env.example .env
```

Open `.env` and fill in the required values:

```bash
# Supabase — from your Supabase project → Settings → API
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key

# Anthropic
ANTHROPIC_API_KEY=sk-ant-...

# Generate these with the commands below
ENCRYPTION_KEY=
API_SECRET=
```

Generate `ENCRYPTION_KEY` and `API_SECRET`:

```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
# run twice — paste first output into ENCRYPTION_KEY, second into API_SECRET
```

## Step 2: Run the database schema

<Warning>
  This step is required. Without it the app will start but nothing will work.
</Warning>

Open your Supabase project → **SQL Editor** → paste and run the contents of [`supabase/schema.sql`](https://github.com/bennetglinder1/proply-crm/blob/main/supabase/schema.sql).

This creates all tables, indexes, RLS policies, and enables the `vector` and `uuid-ossp` extensions.

## Step 3: Install and start

```bash
pnpm install
pnpm dev
```

This starts all four services in parallel:

| Service | URL | What it does |
|---|---|---|
| API server | `http://localhost:3000` | Core REST API, identity resolution, signal ingestion |
| Frontend | `http://localhost:5173` | Web UI |
| MCP server | `http://localhost:3001` | Agent tool interface |
| Worker | — | Background jobs: AI synthesis, sync, scoring |

## Step 5: Connect your AI agent via MCP

Add Proply to your `mcp.json` (Claude Desktop, Cursor, or any MCP host):

```json
{
  "mcpServers": {
    "proply": {
      "command": "npx",
      "args": ["-y", "@goproply/mcp"],
      "env": {
        "PROPLY_API_KEY": "your-api-key",
        "PROPLY_API_URL": "http://localhost:3000"
      }
    }
  }
}
```

Generate your API key under **Settings → API Keys**.
