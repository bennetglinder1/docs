## Prerequisites

Before you start, you'll need:

- [Docker](https://docs.docker.com/get-docker/) installed
- A [Supabase](https://supabase.com) project (free tier works)
- An [Anthropic API key](https://console.anthropic.com) for AI synthesis

---

## Step 1: Clone and configure

```bash
git clone https://github.com/bennetglinder1/proply-crm.git
cd proply-crm
cp .env.example .env
```

Open `.env` and fill in the required keys:

```bash
# Supabase
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_ANON_KEY=your-anon-key
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key

# Anthropic (for AI synthesis and ICP scoring)
ANTHROPIC_API_KEY=sk-ant-...
```

## Step 2: Start everything

```bash
pnpm install
pnpm dev
```

This starts the API server on `http://localhost:3000` and the frontend on `http://localhost:5173`.

## Step 4: Connect your AI agent via MCP

Add Proply to your `mcp.json` (Claude Desktop, Cursor, or any MCP host):

```json
{
  "mcpServers": {
    "proply": {
      "command": "npx",
      "args": ["-y", "@goproply/mcp"],
      "env": {
        "PROPLY_API_KEY": "your-api-key",
        "PROPLY_API_URL": "http://localhost:3000"
      }
    }
  }
}
```

Generate your API key in **Settings → API Keys**.

## Step 5: Make your first query

Once connected, your agent can query any contact with a single tool call:

```
get_contact sarah@acmecorp.com
```

Proply returns the full unified record — every touchpoint, memory, ICP score, and deal stage — from a single call.
