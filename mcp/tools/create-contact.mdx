---
title: "create_contact"
description: "Add a new contact with their full profile"
---

Creates a new contact. Email is required and must be unique within the workspace. Returns the new `contact_id` so you can immediately call `track` or `remember` to log context.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `email` | string | Yes | Email address — must be unique |
| `first_name` | string | No | First name |
| `last_name` | string | No | Last name |
| `company` | string | No | Company name |
| `job_title` | string | No | Job title or role |
| `phone` | string | No | Phone number |
| `linkedin_url` | string | No | LinkedIn profile URL |
| `notes` | string | No | Free-form notes |

## Returns

```
Created contact: Alex Chen
VP of Engineering · Acme Corp
contact_id: a1b2c3d4-e5f6-7890-abcd-ef1234567890
Stage: identified
```

Returns an error if the email already exists in the workspace.

## When to call

Use `create_contact` when you encounter a new person who isn't in the system yet. After creating, call `track` to log how you met them and `remember` to store any facts you already know.
