---
title: "delete_contact"
description: "Permanently remove a contact and all their data"
---

Permanently deletes a contact along with all their activities and memories. This cannot be undone.

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `email` | string | One of | Contact's email address |
| `contact_id` | string | One of | Contact UUID |

## Returns

```
Deleted contact alex@acmecorp.com (a1b2c3d4-...).
```

## When to call

Use for duplicates, test entries, or when a contact explicitly asks to be removed. For contacts you just don't want to actively work — lowering their stage to `identified` is usually better than deleting.

<Warning>
Deletion is permanent. All activities and memories scoped to this contact are removed. Company-level memories are not affected.
</Warning>
