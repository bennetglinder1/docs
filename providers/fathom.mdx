## Connect via API Key

<Steps>
  <Step title="Get your API key from Fathom">
    In Fathom go to **Settings → API** and copy your API key.

    <img width="60%" src="/images/providers/fathom/step-1-api-key.png" alt="Copy Fathom API key" />
  </Step>

  <Step title="Connect in Proply">
    Go to **Settings → Integrations → + Add Integration → Fathom**, paste the key, and click **Connect**.

    <img width="60%" src="/images/providers/fathom/step-2-connect.png" alt="Connect Fathom in Proply" />
  </Step>
</Steps>

## Set up the Webhook (recommended)

<Steps>
  <Step title="Copy your webhook URL from Proply">
    Go to **Settings → Integrations → Webhooks → Fathom** and copy the URL.

    <img width="60%" src="/images/providers/fathom/step-3-webhook-url.png" alt="Copy webhook URL from Proply" />
  </Step>

  <Step title="Add the webhook in Fathom">
    In Fathom go to **Settings → Webhooks → Add Webhook**, paste the URL.

    Fathom uses `whsec_`-prefixed signing secrets. Copy the signing secret and add it to `proply.env`:
    ```bash
    FATHOM_WEBHOOK_SECRET=whsec_your-secret-here
    ```

    <img width="60%" src="/images/providers/fathom/step-4-add-webhook.png" alt="Add webhook in Fathom" />
  </Step>
</Steps>

See the [Fathom webhook reference](/webhooks/fathom) for payload details.
