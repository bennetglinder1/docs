## Overview

Connecting Fathom gives Proply access to your meeting recordings and AI summaries. When a contact is looked up, the agent can retrieve relevant past meetings from Fathom.

For real-time meeting events (logging a `meeting_held` activity automatically when a recording is ready), see the [Fathom webhook](/webhooks/fathom).

## Setup

### 1. Get your API key

In Fathom, go to **Settings → Integrations → API** and generate an API key.

### 2. Connect in Proply

Go to **Settings → Integrations → Fathom**, paste your API key, and click **Connect**.

Proply verifies the key against the Fathom meetings endpoint.

## What this enables

Once connected, the Proply agent can:

- Search past Fathom recordings by contact or company
- Pull AI summaries into contact context
- Retrieve action items from past meetings

## Also set up the webhook

The API key integration is for on-demand lookups. To automatically log meetings as they happen, also configure the [Fathom webhook →](/webhooks/fathom)
