## Overview

Connecting Calendly gives Proply access to your event types and scheduled meetings. This lets the agent answer questions about your scheduling context when working with a contact.

For real-time booking events (automatically creating contacts and logging `meeting_scheduled` when someone books), see the [Calendly webhook](/webhooks/calendly).

## Setup

### 1. Generate a personal access token

In Calendly, go to **Integrations → API & Webhooks → Personal Access Tokens** and generate a new token.

### 2. Connect in Proply

Go to **Settings → Integrations → Calendly**, paste your token, and click **Connect**.

Proply verifies it against the Calendly API and confirms your account name.

## What this enables

Once connected, the Proply agent can:

- Check your available event types
- Query scheduled meetings for a contact
- Pull Calendly context into conversations

## Also set up the webhook

The token integration is for on-demand queries. To automatically create contacts and log bookings as they happen, also configure the [Calendly webhook →](/webhooks/calendly)
