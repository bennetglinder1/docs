---
title: Attio
description: Connect Attio to sync contacts and records into Proply.
---

## Setup

<Steps>
  <Step title="Open API keys in Attio">
    In Attio go to **Settings → API keys** and click **Create API key**.

  </Step>

  <Step title="Name and create the key">
    Give it a name (e.g. "Proply") and click **Create**. Copy the key shown — it won't be shown again.

  </Step>

  <Step title="Connect in Proply">
    Go to **Settings → Integrations → + Add Integration → Attio**, paste the key, and click **Connect**.

  </Step>
</Steps>
