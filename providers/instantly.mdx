## Overview

Proply integrates with Instantly in two ways:

1. **Webhook** — Instantly pushes sequence events (replies, sends, opens, bounces) to Proply in real time
2. **API key** — Proply can query your Instantly campaigns for context

A **reply from a cold email creates a new contact** automatically. This is the core signal — someone replying to your outbound means they're a real prospect.

## API key setup

### 1. Get your Instantly API key

In Instantly, go to **Settings → API Keys** and create a key.

### 2. Connect in Proply

Go to **Settings → Integrations → Instantly**, paste your API key, and click **Connect**. Proply verifies it against the Instantly API.

## Webhook setup

The webhook handles real-time sequence events — replies, opens, bounces.

### 1. Get your webhook URL

```
https://api.yourdomain.com/inbound/instantly/<YOUR_WORKSPACE_ID>
```

### 2. Add the webhook in Instantly

Go to **Settings → Integrations → Webhooks**, add your URL, and select events.

### 3. Set the HMAC secret (production)

```bash
INSTANTLY_WEBHOOK_SECRET=your-signing-secret
```

## Events

| Instantly event | Activity logged | Creates contact? |
|---|---|---|
| `reply_received` | `email_received` | **Yes** — reply = new prospect |
| `email_sent` | `email_sent` | No |
| `email_opened` | `email_opened` | No |
| `email_bounced` | `email_bounced` | No |
| `unsubscribed` | `email_bounced` | No |
