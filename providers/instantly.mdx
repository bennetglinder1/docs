## Connect via API Key

<Steps>
  <Step title="Open API settings in Instantly">
    In Instantly go to **Settings → Integrations → API** and copy your API key.

    <img src="/images/providers/instantly/step-1-api-key.png" alt="Copy Instantly API key" />
  </Step>

  <Step title="Connect in Proply">
    Go to **Settings → Integrations → + Add Integration → Instantly**, paste the key, and click **Connect**.

    <img src="/images/providers/instantly/step-2-connect.png" alt="Connect Instantly in Proply" />
  </Step>
</Steps>

## Set up the Webhook (recommended)

The webhook lets Proply log email events (opens, replies, bounces) in real time as they happen.

<Steps>
  <Step title="Copy your webhook URL from Proply">
    Go to **Settings → Integrations → Webhooks → Instantly** and copy the URL shown.

    <img src="/images/providers/instantly/step-3-webhook-url.png" alt="Copy webhook URL from Proply" />
  </Step>

  <Step title="Add the webhook in Instantly">
    In Instantly go to **Settings → Integrations → Webhooks → Add Webhook**, paste the URL, and select the events you want.

    <img src="/images/providers/instantly/step-4-add-webhook.png" alt="Add webhook in Instantly" />
  </Step>
</Steps>

See the [Instantly webhook reference](/webhooks/instantly) for the full event list.
