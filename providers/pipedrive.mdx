---

## Setup

<Steps>
  <Step title="Open your API settings">
    In Pipedrive go to your avatar → **Personal preferences → API**.

    <img src="/images/providers/pipedrive/step-1-api-settings.png" alt="Open Pipedrive API settings" />
  </Step>

  <Step title="Copy your API token">
    Copy the **Your personal API token** value shown.

    <img src="/images/providers/pipedrive/step-2-copy-token.png" alt="Copy Pipedrive API token" />
  </Step>

  <Step title="Connect in Proply">
    Go to **Settings → Integrations → + Add Integration → Pipedrive**, paste the token, and click **Connect**.

    <img src="/images/providers/pipedrive/step-3-connect.png" alt="Connect Pipedrive in Proply" />
  </Step>
</Steps>
