## Overview

Connecting Fireflies gives Proply access to your meeting transcripts. When a contact is looked up, the agent can pull relevant past meetings directly from Fireflies.

For real-time meeting events (logging a `meeting_held` activity automatically when a recording is ready), see the [Fireflies webhook](/webhooks/fireflies).

## Setup

### 1. Get your API key

In Fireflies, go to **Integrations → API** and copy your API key.

### 2. Connect in Proply

Go to **Settings → Integrations → Fireflies**, paste your API key, and click **Connect**.

Proply verifies the key by calling the Fireflies GraphQL API. If the key is valid, you'll see your Fireflies account email confirmed.

## What this enables

Once connected, the Proply agent can:

- Search transcripts by contact name or company
- Pull meeting summaries into contact context
- Answer questions like "what did we discuss with Acme Corp last month?"

## Also set up the webhook

The API key integration is for on-demand lookups. To automatically log meetings as they happen, also configure the [Fireflies webhook →](/webhooks/fireflies)
