## Connect via API Key

<Steps>
  <Step title="Get your API key from Fireflies">
    In Fireflies go to **Settings → API** and copy your API key.

    <img style={{width:"60%"}} src="/images/providers/fireflies/step-1-api-key.png" alt="Copy Fireflies API key" />
  </Step>

  <Step title="Connect in Proply">
    Go to **Settings → Integrations → + Add Integration → Fireflies**, paste the key, and click **Connect**.

    Proply verifies the key by querying the Fireflies GraphQL API.

    <img style={{width:"60%"}} src="/images/providers/fireflies/step-2-connect.png" alt="Connect Fireflies in Proply" />
  </Step>
</Steps>

## Set up the Webhook (recommended)

The webhook lets Proply log a meeting activity on contact timelines as soon as a transcript is ready.

<Steps>
  <Step title="Copy your webhook URL from Proply">
    Go to **Settings → Integrations → Webhooks → Fireflies** and copy the URL.

    <img style={{width:"60%"}} src="/images/providers/fireflies/step-3-webhook-url.png" alt="Copy webhook URL from Proply" />
  </Step>

  <Step title="Add the webhook in Fireflies">
    In Fireflies go to **Integrations → Webhooks → Add webhook**, paste the URL, and select **Transcript Ready**.

    <img style={{width:"60%"}} src="/images/providers/fireflies/step-4-add-webhook.png" alt="Add webhook in Fireflies" />
  </Step>
</Steps>

See the [Fireflies webhook reference](/webhooks/fireflies) for payload details.
