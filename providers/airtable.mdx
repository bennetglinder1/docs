## Overview

Airtable is a CRM import source. You connect once via OAuth, then import contacts from a selected base and table. After import, Proply does not keep a live sync with Airtable — changes in either direction do not automatically propagate.

## Setup

### 1. Connect Airtable

In your Proply dashboard, go to **Settings → Integrations → Airtable** and click **Connect**. You'll be redirected to Airtable to authorize access.

### 2. Select your base and table

After connecting, select the Airtable base and table that contains your contacts.

### 3. Run the import

Click **Import** to pull contacts into Proply. Deduplication runs on email address — existing contacts are merged, not duplicated.

<Note>
  Proply strips placeholder emails injected by Airtable (e.g. `user@airtable.import`) and skips those rows.
</Note>

## Field mapping

Proply maps these Airtable fields automatically:

| Airtable field | Proply field |
|---|---|
| Name / Full Name | `full_name` |
| Email | `email` |
| Phone | `phone` |
| Company | `company` |
| Title / Job Title | `title` |
| LinkedIn URL | `linkedin_url` |
| Website | `website` |

## Environment variables

| Variable | Description |
|---|---|
| `AIRTABLE_CLIENT_ID` | OAuth app client ID |
| `AIRTABLE_CLIENT_SECRET` | OAuth app client secret |
| `AIRTABLE_REDIRECT_URI` | Must match the redirect URI in your Airtable OAuth app |
